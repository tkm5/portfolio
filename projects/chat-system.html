社内向けチャットシステム開発

欧州系コンサルティングファーム | 社内プロジェクト | 2022年2月 - 2025年4月

概要

Azureのクラウド環境を基盤とし，生成AIおよびRAG (Retrieval-Augmented Generation) 技術を活用した，社内問い合わせ対応チャットシステムを開発・実装．

詳細

Azure AI Search，Azure OpenAI Serviceを駆使し，高度な自然言語処理による高精度な回答生成を実現．さらに，Azure DevOpsを用いたCI/CDパイプライン構築を提案・推進し，開発プロセスの効率化に貢献．

成果

これらの実績が評価され，2024年度コーポレートアワードを受賞．

使用技術

Azure Azure OpenAI Service Azure AI Search Azure DevOps RAG CI/CD
